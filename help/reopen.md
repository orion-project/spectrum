# Reopen Graph

```
► Graph ► Reopen
```

Read graph points from a different data source (e.g., data file), apply the same transformations, and replace the current graph.

Depending on the graph's data source, the command can be treated differently. For example, for file-based data sources, it shows  a dialog for selection of a new file. For [formula](./add_formula.md) data sources, it opens a formula editor dialog and allows for changing the formula code.
